Parkhaus Schranken

Dieses Parkhaus-Schranken-Programm dient dazu, Schranken in einem Parkhaus zu steuern (in Simulation). Es erfasst beim Öffnen und Schließen der Schranken die einfahrenden und ausfahrenden Besucher oder Mieter und registriert diese Informationen. Das Programm kann an verschiedene Parkhäuser angepasst werden, indem man die Einstellungen entsprechend anpasst.

Es erstellt automatisch ein Protokoll mit dem Dateinamen "Protokoll.txt", das im selben Ordner erscheint. Das Programm ist in der Lage, die Einstellungen zu speichern und beim nächsten Verwenden wieder einzulesen, was in der Datei "Einstellungen.txt" gespeichert wird, die ebenfalls automatisch im selben Ordner erzeugt wird.

Tasten:

1 Start: Das Programm liest die alten Daten ein und startet sofort das Hauptprogramm. Um das Hauptprogramm zu beenden, drücken Sie die Taste "0".
2 Einstellungen: Hier können die Einstellungen für die Anzahl der Parkplätze vorgenommen werden. Diese werden in der Datei "Einstellungen.txt" gespeichert. Wenn die Einstellungen vorgenommen wurden, startet das Hauptprogramm.
0 Beenden: Diese Taste beendet das Programm.
